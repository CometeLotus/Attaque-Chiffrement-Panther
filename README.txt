Attacks on Panther

This Python project implements the "Panther" cipher and explores various attacks on the cipher.



Running the Project
To run the code, follow these steps:

1- Open a command prompt or terminal window.
2- Navigate to the directory containing the Panther.py file.
3- Enter the command python Panther.py to run the program.

After running the program, you should see the test results displayed in the terminal window.

Note that the same key, IV, and plaintext are used every time the program is run. Instead, random generators can be used to generate different values each time.
